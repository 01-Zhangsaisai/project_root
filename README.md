# 文件解析器项目

## 项目简介

这是一个用于解析多种文件格式的 Python 项目，包括 PDF、DOCX、DOC、HTML 和 DJVU。该项目提供了简单易用的接口来提取文本、元数据和图像。

## 特性

- 支持 PDF 文件解析
- 支持 DOCX 和 DOC 文件解析
- 支持 HTML 文件解析
- 支持 DJVU 文件解析
- 提供详细的错误处理和异常管理
